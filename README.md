# Instalação

```go 
go get github.com/marcelo68ms/gofinance
```

# Exemplos

### Obter somente cotações de moedas e bolsas.

```go
import (
	"encoding/json"
	"fmt"
	"github.com/marcelo68ms/gofinance"
)

func main() {

	quotations, err := gofinance.GetQuotations("KEY")
	if err != nil {
		fmt.Println(err)
		return
	}

	json, err := json.Marshal(quotations)
	if err != nil {
		fmt.Println(err)
		return
	}

	fmt.Printf("%s\n", json)
}
```
